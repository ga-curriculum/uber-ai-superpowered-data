## About

This module explores the crucial mental shift from deterministic code (guaranteed outputs) to probabilistic AI, where success comes from "steering" models rather than simply writing code.

## Module Outcomes
- Differentiate between deterministic software engineering and probabilistic AI engineering.
- Diagnose AI system failures where the code is functional but the output is not useful or reliable.
