## About

This module introduces the key architectural patterns for single AI agents. Learners will move beyond theory in a scenario-based exercise, analyzing several business problems to select the most appropriate and efficient architecture for each.

## Module Outcomes 
- Identify the core components of a single AI agent (e.g., orchestrator, tools, memory).
- Describe common architectural patterns for agents, including agents with tools and those using Retrieval-Augmented Generation (RAG).
