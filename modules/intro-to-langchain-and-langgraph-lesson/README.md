## About

This module introduces LangChain and LangGraph as frameworks for building LLM-powered systems. Learners practice building simple but auditable workflows, progressing from narrative branching in LangChain to stateful routing and typed classifiers in LangGraph.

## Module Outcomes

- Compare LangChain’s role in prototyping with LangGraph’s design for production workflows.
- Apply LangGraph primitives (nodes, edges, state, graph) to design transparent, auditable systems.
- Build workflows that evolve from prompt-based narratives to stateful routing and structured classifiers.
