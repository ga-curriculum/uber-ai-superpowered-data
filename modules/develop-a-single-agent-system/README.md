## About

Now let's put all the pieces together. Participants will decompose tasks and select models to assemble a functional single agent. This two-part lab series builds a **SOX Audit Copilot**: an AI assistant that helps finance teams generate and validate audit evidence for SOX controls.

You'll combine deterministic checks (Python logic) with generative AI (LLM) to produce audit-ready outputs, then add observability and evaluation to ensure reliability.

## Module Objectives

- Develop a functional single agent by assembling core components, including an orchestrator and deterministic tools.
- Test the final agent's performance against a set of requirements and justify the design choices made during its construction.
- Capture full execution traces with LangSmith for debugging and verification.
- Create evaluation datasets and compute quality signals to iterate with confidence.
